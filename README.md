# LearnWise AI 🎓

A personalized learning platform that recommends the right resources based on how you learn.

## Features
- AI-powered resource recommendations
- Smart filters by topic, format & difficulty
- Custom cursor animation
- Learning path builder
- Multi-dimensional resource ratings
